## Summary: when multiple competition_ids exist

- **Grand Slams:** 5 IDs each (Men/Women Singles, Men/Women/Mixed Doubles).
- **Masters 1000:** 2–4 IDs each (ATP Men Singles/Doubles; often WTA Women Singles/Doubles). Paris and Monte Carlo are ATP-only in this list; Shanghai is ATP-only.
- **Canada:** Multiple IDs because Toronto and Montreal alternate; use the four main-draw IDs listed.
- **ATP Finals / WTA Finals:** 2 IDs each (Singles and Doubles). Next Gen ATP Finals is a different competition (14754).
- **Olympics:** 5 IDs (all five disciplines).
- **Davis Cup / BJK Cup:** 1 ID each.
- **500s:** 2 or 4 IDs depending on whether WTA is present; Queen's has no main-draw match in this file.

All IDs were derived from **sr_seasons.json**; season coverage is mostly 2024–2026 for main-draw events.
